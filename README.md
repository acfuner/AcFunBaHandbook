###【账号问题】

1，如果你的帐号激活不能，或者是邮箱有误，或者是你觉得你被误封，请[荐!]用QQ联系客服：   

<img src="pics/kefu.png" width="560" />  

与 <a href="mailto:report@acfun.tv">report@acfun.tv</a> 邮件联系亦可。账号问题可以加入官方账号处理群：246866915

2，账号找回收不到邮件：目前的账户名已无法使用，如果收不到邮件请首先检查一下找回密码的用户名那里是否填的是昵称，否则不能收到邮件。

3，登陆密码错误：  
目前的账户名已无法使用，首先请检查账号是否填写的是 昵称 或者是 邮箱地址 。

如果尝试两次之后依然无法登陆，请及时联系客服！尝试三次都无法登陆会造成【密码登陆错误次数过多】。

4，密码登陆错误次数过多：如果登陆的时候提示这个消息，那么1天之后才能继续登陆，切换ip之后刷新CD。

5，有一段时间没登陆，登陆后发现被封一百年：  
提示为：您的账号因为存在安全风险而被限制使用，请加入账号处理群或联系客服解决。  
提示为：发布违规合辑，如果你不是广告营销号，也从来没创建过合辑，那么就应该是被盗号了，请联系客服，在处理完成之后请修改密码。  

6，邀请码被盗领：请在邀请码没过期之前请第一时间用掉它，防止被别人卖掉，并及时修改密码和联系客服。

###【删帖标准】


1，不要在本吧除了专门的求邀请码的帖子以外的地方求、送邀请码，否则会被删封。带原创作品求码的除外。  
带原创作品的up主快速注册通道：http://www.acfun.tv/info/#page=fastpath  
A吧永久求邀请码专贴链接：http://ask.acfuner.com/invite  


2，有关于主站的任何问题，请找客服。对主站封禁情况不满，请找客服。嘲讽，辱骂或有其他不理智行为，删封。  

3，无意义水贴，15字，小尾巴影响到吧友正常浏览贴吧，删帖。如果是用机器自动回复或者发广告，封禁。  

4，请不要“自顶帖子”或者用“留邮箱的方式骗回复”或发帖时图文无关（即放镇楼图），否则删帖。  

5，请不要挖坟，挖坟的账号一天，被挖坟的帖子有可能会被删除。  

6，莫谈B事/B吧事，否则有可能被删帖。  

7，日经/周经/月经贴有可能被删除。  

8，骂人/低俗/政治/无故嘲讽、诋毁、诽谤吧友都会被删帖或封禁。
